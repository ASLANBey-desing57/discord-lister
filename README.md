# Discord-Listing
An all-in-one poster for bot stats. 

### Example
``` js
const botlist = require('Discord-Listing');

let settings = {
  discordbotsorg: true,
  discordbotsorg_token: 'Set Token Here',
  discordbotsgg: true,
  discordbotsgg_token: 'Set Token Here',
  servercount: 0,
  shardscount: 0

botlist.post(settings)
```
